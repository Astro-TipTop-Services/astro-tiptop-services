## Conversion from Magnitude to Number of Photons per Subaperture per Frame {#mag-to-photons}
The value of the parameter `NumberPhotons` (expressed in photons per subaperture per frame) for a natural guide star of a given magnitude is calculated using the following formula:

$$
\boxed{
\text{NumberPhotons}
=
ZP \Big(\frac{Tot_{\mathrm{throughput}}}{\mathrm{Sensor}_{\mathrm{framerate}}}\Big)\;
\Big(\frac{A_{\mathrm{pupil}}}{N_{\mathrm{sa,tot}}}\Big)\;
10^{-0.4\,Mag}
}
$$

with

$$
A_{\mathrm{pupil}}=\pi\Big(\tfrac{D}{2}\Big)^2\bigl(1-\mathrm{ObscurationRatio}^2\bigr),
\qquad
Mag = Mag_V + \mathrm{extinction\_correction}.
$$

#### Effective number of subapertures

Let $N_{lenslet}$ be the number of lenslets across the pupil (square grid). 
The total **effective** number of illuminated subapertures is

$$
N_{{sa,tot}}=
\begin{cases}
N_{lenslet}^2, & \text{if } N_{lenslet}\le 2,\\[4pt]
\left\lfloor N_{lenslet}^2 \,\tfrac{\pi}{4}\,\bigl(1-\mathrm{ObscurationRatio}^2\bigr)\right\rfloor, & \text{if } N_{lenslet}>2~,
\end{cases}
$$

which approximates the circular pupil (disk-in-square factor $\pi/4$) and the central obscuration $(1-\epsilon^2)$, then floors to an integer.

#### Spectral band extinction correction
For a **spectral class M0V star**, the apparent magnitude is corrected by an extinction term that varies by band:
$$
Mag = Mag_V + \text{extinction\_correction}
$$
<div style={{ display: 'flex', justifyContent: 'center'}}>
|                       | B band     | V band    | R band   | I band    | J band   | H band  | K band  |
|-----------------------|:----------:|:---------:|:--------:|:---------:|:--------:|:-------:|:-------:|
| $\text{extinction\_correction}$| $1.37$ | $0$ | $-1.26$ | $-2.15$ | $-2.49$ | $-3.03$ | $-3.29$ |
</div>
<div style={{ textAlign: 'justify'}}>
This formula converts the visual magnitude, corrected for spectral band extinction, into the expected number of photons detected per subaperture per frame. 
It takes into account instrumental parameters such as the telescope’s effective collecting area, the number of wavefront sensor lenslets, the total system throughput, and the wavefront sensor's frame rate.
</div>
